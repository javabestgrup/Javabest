# ☪️ JAVAISLAMI — Modern Islamic Web App

[![Web3-Ready](https://img.shields.io/badge/Web3-Ready-blue.svg)](https://github.com/Brodin)
[![Security-Focused](https://img.shields.io/badge/Audit-Security-gold.svg)](https://github.com/Brodin)
[![AdSense-Optimized](https://img.shields.io/badge/AdSense-Optimized-green.svg)](https://github.com/Brodin)

**JAVAISLAMI** adalah aplikasi panduan Islam modern berbasis web yang menggabungkan estetika desain minimalis dengan fungsionalitas tinggi. Proyek ini dibangun sebagai bagian dari portofolio profesional di bidang **Web3 Development** dan **Security Auditing**.

---

## 🚀 Fitur Utama

- **Real-time Prayer Schedule:** Penentuan waktu shalat otomatis berdasarkan koordinat GPS pengguna.
- **Interactive Al-Qur'an:** Fitur pelacakan progres bacaan (Reading Progress) yang responsif.
- **Smart Qibla Finder:** Kompas presisi dengan animasi halus untuk menentukan arah Ka'bah.
- **Islamic Business Community:** Ruang *networking* untuk ekosistem ekonomi syariah dan UMKM halal.
- **Islamic Law (Fiqih):** Panduan hukum Islam 4 mazhab yang terstruktur untuk kemudahan akses informasi.

---

## 🛠️ Stack Teknologi

Proyek ini menggunakan pendekatan **"Zero Bloat"** untuk memastikan kecepatan muat (loading speed) yang maksimal, yang sangat krusial untuk persetujuan **Google AdSense**.

- **Frontend:** HTML5 (Semantic), CSS3 (Custom Variables), JavaScript (ES6+).
- **Design System:** Minimalis Modern dengan palet warna Biru Navy (#1a4a7a), Emas, dan Putih.
- **Icons & Fonts:** Google Fonts (Poppins, Scheherazade New) & Unicode Icons.
- **Optimization:** Pemisahan file CSS & JS untuk mempermudah audit keamanan (Security Audit).

---

## 🔒 Keamanan & Web3 Roadmap

Sebagai pengembang yang berfokus pada **Security Auditor**, aplikasi ini dirancang dengan prinsip:
1. **Clean Code:** Memudahkan proses *debugging* dan identifikasi celah keamanan.
2. **Privacy First:** Tidak menyimpan data sensitif pengguna di server tanpa enkripsi.
3. **Web3 Integration (Future):** Implementasi pembayaran Zakat/Infaq menggunakan Smart Contract (Solidity) dan konektivitas Wallet.

---

## 📈 Strategi Monetisasi

Aplikasi ini disiapkan untuk:
- **Google AdSense:** Struktur konten yang SEO-friendly dengan slot iklan strategis.
- **Affiliate Marketing:** Rekomendasi produk halal di fitur Marketplace.
